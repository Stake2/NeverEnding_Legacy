Speedy

URL to paste:
https://cdn.jsdelivr.net/gh/MagixModLegacy/Magix@master/Speedy.js

Speedy is a mod that expands your choice of fast forward mode.
Click on special button and type a number in a prompt to speed up the game at desired number.
You can now decide at which speed the game will run. Say goodbye to choice between pause, x1, x30
Now you can pick: 0.5x, 4x, 8x... Any speed you desire
Note that it will consume fast ticks.
